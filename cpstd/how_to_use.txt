LINUX
==============================================================================
1. Go to the project folder

2. Execute this command:

'
cmake -B build -DCMAKE_INSTALL_PREFIX=/usr/local && cmake --build build && sudo cmake --install build
'

Note:

Do 'sudo rm -rf /usr/local/include/cpstd' if you update the lib so you need to remove the old version before building the newer one

3. Done you now should be able to include any header of the cpstd like this:

'
#include <cpstd/cpvec.h>
'
